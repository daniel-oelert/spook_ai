# Spook AI

Spook AI is a VS Code extension that implements a Agent Command Environment for agentic coding, using Pyodide to run sandboxed Python code for recursive language model calls. Utilize your existing workspace setup to grant your agent access to your files, terminals, and other VS Code features like LSP and code actions.

# Features

-   Create, fork, edit and delete sessions
-   Manage agents and tools
-   Use any LLM provider
